# Look-and-Say 수열

문제 속 수열은 이전 결과값을 통해 현재 결과값을 얻으며, 이전 결과값들에 연쇄 의존한다고 이해했습니다.
따라서, 결과를 얻는 과정을 세 단계의 함수로 분리하여 연산하였습니다.
분리한 설계와 근거는 아래와 같습니다.

| 함수명          | 기능                                     | 설명                                                |
| --------------- | ---------------------------------------- | --------------------------------------------------- |
| `calculateNext` | 한 항(prev)을 읽어 **다음 한 항**만 생성 | 순수 변환 규칙. 가장 작은 단위라 단독 테스트가 쉽다 |
| `calculateLn`   | L1부터 **n번 순회**하여 Lₙ 도달          | 1장의 결론(순회 필요)을 그대로 구현                 |
| `getCenter`     | Lₙ에서 **중앙 두 글자** 추출             | 요구사항(출력)에 해당하는 마지막 단계               |
